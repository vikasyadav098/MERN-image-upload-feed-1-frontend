# 🚀 MERN Image Upload Feed — Frontend ⚛️

A clean and responsive React frontend for the MERN Image Upload Feed app.  
Users can create posts with images and view them in a dynamic feed.

---

## ✨ Features

- 📝 Create Post with Image Upload
- 📰 Live Feed from API
- 🔗 Axios-based API integration
- 🌐 Environment-based API URL (Vite)
- 📱 Fully Responsive UI
- 🧭 React Router Navigation
- ☁️ Deployed on Vercel

---

## 🛠️ Tech Stack

- ⚛️ React (Vite)
- 🔗 Axios
- 🧭 React Router
- 🎨 CSS
- ☁️ Vercel (Frontend Hosting)
- 🚀 Render (Backend API)
